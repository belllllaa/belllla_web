## 第11课时专节：研报复现 · 换手率与价量（依据 `经典研报复现.md`）

### 1. 课程定位

- **研报驱动**：从「摘要 → 因子定义 → 假设 → 回测结论」抓 20% 核心内容即可。  
- 三个层次：**指标因子化** → **迭代优化** → **多因子融合**（融合方式错误会劣于单因子）。

### 2. 换手率因子系列（东方证券技术分析因子系列）

| 代际 | 名称 | 核心假设 | 要点 |
|------|------|----------|------|
| 一代 | 传统换手率（量小） | 换手越低未来收益越好（冷门/便宜） | 全 A 月度 IC 约 -7.5% 量级（注意报告口径与实现差异）；**十分组单调性**可能一般；偏**空头有效**。 |
| 二代 | 量稳换手率 | **稳定性**比绝对大小更重要：20 日换手标准差越小越稳 | 常做**市值等风格中性化**（回归残差）；与一代相关性可 >90%，同用需**正交化**。 |
| 矛盾 | 双重分组 | 理论「量稳好且量小好」——实证在**量稳好**组内**量大**反而更好 | 案例：持续上涨需**持续成交量**；量稳与量小假设在不同子样本上矛盾。 |
| 三代 | U 加法（优加换手率） | **分层**：量稳前 50% 内按量**从大到小**打分；量稳后 50% 内按量**从小到大**打分；再与量稳排名合成 | 避免简单线性加总；绩效上常优于单因子与简单相加。 |

**风格中性化（再强调）**：因变量 = 目标因子，自变量 = 市值等风格因子，**残差**为中性化后因子。

**量纲**：多因子相加前宜 **截面标准化**（(x−μ)/σ）等，避免量纲混加。

**样本内外**：按**时间**切分，严禁随机打乱；样本外用于验证，防过拟合。

### 3. 价量相关性因子（海通思路）

- **定义**：滚动窗口内（如 20–60 日）**收盘价序列与换手率序列的皮尔逊相关系数**。  
- **含义**：正相关 → 价量同向居多；负相关或弱相关 → 背离或趋势不稳。  
- **特点**：可能**多头弱、空头强**；需关注**市值暴露**（大票价量相关往往更高）。  
- **可扩展**：分钟级价量再降频到日频，丰富日内信息。

### 4. 实践提醒

- **线性模型**：因子质量优先时，**线性合成**往往已很强；复杂模型是增量。  
- **读研报**：区分**绝对收益**与**超额收益**；关注基准与市场环境。  
- **流动性**：换手类因子本质是**流动性**研究，在不同市场流动性结构下表现会不同。

### 5. 概念速查（与课程一致）

| 概念 | 说明 |
|------|------|
| 量小因子 | 换手率本身，越低越「量小」 |
| 量稳因子 | 20 日换手标准差，越小越稳 |
| U 加法 | 按量稳分层后，对量小方向分层处理再合成 |
| 截面标准化 | 逐日 (x−μ)/σ |
| IC | 因子值与未来收益的相关系数 |
| 空头因子 | 识别下跌强、上涨弱 |

## 与本地工程的关系

- **QMT 策略**（`qmt/回测策略/`）：适合快速验证行情与交易接口；**行业/财务深度**常需外接数据。  
- **IDE + Backtrader / 自研回测**（课程第4、7、8课时方向）：与 `belllla_web` 仓库中 Python 量化脚本可同构演进。  
- **因子预处理与中性化**（第10、12课时）与 **凸优化组合**（你笔记中的均值方差/约束）可衔接：先得到干净因子截面，再进组合优化。
