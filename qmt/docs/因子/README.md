# 因子知识库

本目录集中存放**因子相关**的表格与脚本，与 `docs/` 下的函数知识库（QMT 知识库等）并列。

## 本目录文件

| 文件 | 说明 |
|------|------|
| **factor_reference.xlsx** | 因子参考表 Excel：大类、细分因子名、含义、2024/2025 表现、排名、策略建议等，便于在 Excel 中查看与编辑。 |
| **export_factor_reference.py** | 生成上表的脚本。修改脚本内 `ROWS` 后，在本目录执行 `python export_factor_reference.py` 可重新生成 xlsx。 |
