## 3. 分课要点摘要

### 第1课 · 量化入门与全流程

- **目标**：打破「黑箱」印象，建立从数据到实盘的**全局地图**。
- **案例**：西蒙斯 / 文艺复兴 — 数学驱动、纪律执行、模型至上、高频统计优势。
- **岗位参照**（课程表）：量化研究员、量化工程师、风控、公募量化、金融科技等（技能：Python、统计、金融、工程/深度学习等）。

### 第2课 · Python 基础（2.1–2.5）

- **2.1 语法**：脚本能跑、模块组织、为后续数据处理打底。
- **2.2 Pandas**：索引、`groupby`、合并；**禁止**在大表上用 `iterrows` 做重逻辑。
- **2.3 时序**：`resample`、滚动、`shift` 对齐；与因子滞后、T 日因子对 T+1 收益严格对齐。
- **2.4 可视化**：净值、回撤、分组收益、IC 分布等。
- **2.5 OOP**：策略类、参数、状态、日志，便于回测与实盘复用。

### 第3课 · 行业全景 + 代码 Review

- **Code Review 动机**：资金安全、回测速度、可维护迭代。
- **Pythonic**：推导式、`sum(..., generator)`；筛选用布尔索引 / `query()`。
- **向量化示例**：金叉用 `shift(1)` 错位比较整列，避免逐行 `for`。
- **工程化**：Type Hints、`try/except`、Docstring、语义化命名、PEP 8。
- **行业地图**：量化私募（绝对收益、Alpha、2/20）、公募（相对收益、合规与跟踪误差）、自营/HFT（技术驱动）。
- **私募现实**：Alpha 衰减、竞争与研发强度。

### 第4课 · Backtrader + 海龟

- **Backtrader**：`Cerebro`、数据源、`Strategy`、佣金/滑点、分析器。
- **海龟**：突破系统；课程作业对比短线 vs 长线系统 — **收益高未必夏普高**；盈亏比与胜率分离。
- **MACD 规范**：优先 `bt.ind.MACD` + `CrossOver`，避免手算与浮点比较隐患。

### 第5课 · 绩效评估与策略拆解

- **收益**：总收益 vs **年化 CAGR**；跨策略比较用年化。
- **基准**：沪深300 / 中证500 / 中证1000 / 无风险利率等；注意「市场涨 20%、策略 15%」→ **超额为负**。
- **风险**：波动率（日频年化 ×√252）、**最大回撤 MDD**、**卡玛** = 年化收益 / |MDD|。
- **夏普**：(Rp−Rf)/σ；警惕 Rf 乱取、样本过短、非正态、**Sharpe 虚高（过拟合）**；可配合 **PSR** 等。
- **其他**：Sortino（下行波动）、**信息比率**（相对基准的超额/跟踪误差）。
- **交易质量**：期望 = 胜率×平均盈利 − (1−胜率)×平均亏损；趋势策略可低胜率高盈亏比。
- **成本**：利润因子、换手率；佣金、滑点、冲击成本 — **回测必须计入**。
- **评价原则**：多指标、样本外、压力测试、成本真实、定期复核。
- **实务**：风险加倍可能因 **波动率拖累**、回撤加深导致长期收益反而下降（海龟作业）。

### 第6课 · 因子理论 / 聚宽

- **因子定义**：解释截面收益差异的变量；多因子 = 多标准综合打分。
- **有效逻辑**：风险补偿、行为非理性、制度约束（融券、容量等）。
- **经典分类**：价值（PE/PB）、质量（ROE）、成长（营收增长）、动量、波动率等。
- **6.2–6.3**：多因子合成、聚宽云端回测与本地思路对应。

### 第7–8课 · 本地回测框架

- **Backtrader vs 本地**：单策略/技术指标 vs **全市场截面、多因子**；事件驱动 vs **批量向量化**。
- **五层架构**：Data（行情/涨跌停/行业）→ Factor → Signal（排序、Top N）→ Trading（持仓、成本）→ Evaluation（收益、夏普、回撤、IC）。
- **落地顺序**：DataLoader → 动量等单因子 → SignalGenerator → 主程序串联单因子回测。

### 第9课 · 中期复盘

- 从「策略」抽象到「因子」、迭代路径（与第10–12课衔接）。

### 第10课 · 因子预处理与中性化

- **目录约定示例**：`data_*`、`factors/raw|filtered|preprocessed|industry_neutralized|neutralized`、`outputs`。
- **可交易过滤**：停牌、涨跌停、ST；流动性：低价、低成交额、低换手等；样本 = 因子宇宙 ∩ 可交易宇宙。
- **Pipeline**：去极值（Winsorize / 百分位 / MAD / IQR）→ 缺失填充（**行业中位数**优于乱填 0）→ 标准化（Z-Score 等）。
- **标准化对比**：Z-Score（Pearson、保留幅度）vs Rank（Spearman、稳健）；ML 场景常 Rank，传统多因子回归常 Z-Score。
- **行业中性**：组内 Z-Score vs **行业哑变量回归残差**。
- **Barra 思路**：收益分解为风格暴露 + 特异 α；**低 PE 因子若 R² 高、Size 暴露大**，可能是风格而非 α — **回归中性化**剔除。
- **选用**：资源少可先行业中性；产品级/对标基准再 Barra；可先行业快速验证再 Barra 精细化。

### 第11课 · 研报复现

- **完整展开见** [`阿波量化知识库.md`](./阿波量化知识库.md)（换手率系列、U 加法、价量相关、样本内外等）。

### 第12课 · IC 与多因子评价

- **对齐**：T 日因子 vs **T+1 收益**；Spearman vs Pearson。
- **IC 序列**：均值、标准差、**IC_IR** = Mean_IC / Std_IC、胜率、t 值。
- **Mean IC 粗判**：>0.05 强；0.02–0.05 中；0–0.02 弱；需结合换手与显著性。
- **诊断**：IC 时序、滚动 IC、衰减；U 型因子 IC 可能失效 — 需散点/互信息。
- **分组**：十分组收益、单调性、长短组合。
- **体系**：IC 质量 + 稳定性 + 单调性 + 换手（成本）。

### 第13课 · 多因子合成与实盘

- **合成前**：截面 z-score；权重仅用**历史窗口**估计，**禁未来函数**。
- **权重**：等权、IC 加权、IC_IR 加权、收益加权等；权重和为 1、处理 NaN、记录窗口。
- **优化目标**：多目标 — IC/IR、OOS IC、收益、换手、稳定性。
- **约束**：行业/风格中性、单因子权重上限、换手与成本惩罚、可交易性。
- **线性模型权重**：Ridge/Lasso；**滚动训练、时间切分、禁随机打乱**；VIF 看共线性。
- **非线性**：XGBoost/LightGBM；仍须**时间切分**；可结合特征重要性/SHAP。

## 4. 开发检查清单（自研 Python 策略）

1. **数据**：复权、停牌、涨跌停、ST、财务披露时点；避免前视偏差。  
2. **因子**：去极值 → 缺失 → 标准化 → 行业/Barra 中性（按需求）。  
3. **标签**：T 因子 vs T+1 收益对齐；**时间序列切分**样本内外。  
4. **评价**：IC/IC_IR、分组、单调性、换手与成本后收益。  
5. **回测**：成本、滑点、冲击；幸存者偏差；多指标与压力测试。  
6. **实盘**：信号 → 风控 → 下单接口（可与 QMT 等对接）。
