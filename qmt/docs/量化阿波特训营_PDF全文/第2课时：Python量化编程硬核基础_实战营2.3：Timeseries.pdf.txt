1. 设定窗口大小（如5天）
2. 从第一个位置开始计算
3. 窗口向后滑动一格
4. 重复计算直到结束

布林带（Bollinger Bands）
波动率指标
布林带组成：
- 中轨 - N日移动平均（通常20日）
- 上轨 - 中轨 + 2倍标准差
- 下轨 - 中轨 - 2倍标准差
计算公式：
含义：
- 价格通常在上下轨之间波动
- 触及上轨 → 可能超买
- 触及下轨 → 可能超卖
- 轨道宽度反映波动程度
- 中轨 = MA(20)
- 上轨 = MA(20) + 2 * STD(20)
- 下轨 = MA(20) - 2 * STD(20)
