===== 第 4 页 =====

五大模块架构
分层设计理念
今天重点： 
前三层（Data + Factor + Signal）
┌─────────────────────────────────────────────┐
│           本地回测框架                                                                                   │
├─────────────────────────────────────────────┤
│  1. Data Layer      (数据层)                                                                         │
│     └─ 读取价格、收益率、涨跌停、行业数据                                           │
│                                                                                                                   │
│  2. Factor Layer    (因子层)                                                                        │
│     └─ 计算并保存因子值（动量、反转、量价）                                       │
│                                                                                                                   │
│  3. Signal Layer    (信号层)                                                                        │
│     └─ 因子 → 排序 → 选股 → 交易信号                                                   │
│                                                                                                                   │
│  4. Trading Layer   (交易层)                                                                       │
│     └─ 持仓管理、调仓、成本计算                                                             │
│                                                                                                                   │
│  5. Evaluation Layer (评估层)                                                                    │
│     └─ 收益率、夏普、最大回撤、IC等指标                                               │
└─────────────────────────────────────────────┘
