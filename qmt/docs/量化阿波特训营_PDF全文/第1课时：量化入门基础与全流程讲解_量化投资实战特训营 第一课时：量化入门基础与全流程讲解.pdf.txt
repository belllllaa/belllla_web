股票定价理论：PE、DCF与市场预期
1. 市盈率 (PE Ratio)
PE = 股价 / 每股收益 (EPS)
核心含义：以当前盈利水平，回收投资成本所需的年数。
市场预期：PE越高，代表市场对未来增长的预期越乐观（或存在泡沫）。
相对估值：需对比行业平均水平及公司历史PE中枢。
量化应用： 价值因子（Value Factor）的核心指标，用于识别被低估的资产。
2. 现金流折现 (DCF)
V = Σ [FCFₜ / (1+r)ᵗ]
核心思想：公司价值等于其未来所有自由现金流的现值之和。
关键变量：自由现金流(FCF)、折现率(r)、永续增长率(g)。
绝对定价：理论上最严谨的定价模型，反映公司内在价值。
量化应用： 帮助识别基本面重大变化，避免陷入“估值陷阱”。
“量化投资者理解基本面定价，是为了识别高溢价是合理增长还是纯粹泡沫。”

期权市场：权利、义务与BS定价模型
看涨期权 (Call Option)
实例： 行权价110元，权利金2元。
• 股价涨至120元：行权获利 120 - 110 - 2 = 8元。
• 股价跌至90元：放弃行权，损失 2元（权利金）。
核心：锁定最高成本，收益理论无限。
看跌期权 (Put Option)
实例： 行权价90元，权利金1元。
• 股价跌至80元：行权获利 90 - 80 - 1 = 9元。
• 股价涨至110元：放弃行权，损失 1元（权利金）。
核心：作为“保险”，对冲资产下跌风险。

衍生品风险：波动率、凸性与对冲失效
波动率 (Volatility)
• 隐含波动率 (IV)： 市场对未来波动的定价。IV上升，期权变贵。
 • 历史波动率 (HV)： 基于过去价格计算的真实波动。
 • 风险点： 卖出期权者在波动率骤增时（如VIX跳升）面临巨大亏损。
凸性效应 (Convexity)
• 非线性收益： 价格变动与收益不成正比。大幅波动时，期权收益超
过线性预期。
 • Gamma风险： 衍生品交易员追求凸性带来的超额收益，但也需承担
价格剧烈跳空的风险。
为什么极端市场下对冲可能失效？
关联性坍塌
在极端恐慌中，原本负相关的资产（如股债）可
能同步暴跌，对冲逻辑彻底失效。
流动性枯竭
对冲工具无法按理论价格成交，甚至出现无人接
盘的局面，导致无法及时止损。
对手方风险
如2008年金融危机，即使你买了保险（CDS），如
果保险公司（雷曼）倒闭，对冲也无意义。
警惕：对冲不是万能药，极端市场下现金与流动性才是最终防线。

A股特色规则：T+1 交易制度的约束与应对
核心约束：T+1
当天买入的股票，次日才能卖出。这意味着一旦买入决策错误，
当天无法通过反向操作止损，必须承担隔夜风险。
直接限制了纯粹的日内高频交易策略在A股的实施。
1. 截面策略 (Cross-sectional)
相比于预测单只股票的时序走势，量化更倾向于在全市场范围内寻找相对
强弱，通过持有组合来对冲单只股票的波动风险。
2. 底仓 T+0 策略
通过持有长期底仓，利用日内波动进行高抛低吸。卖出的是旧底仓，买入的
是新仓位，从而在规则内实现日内交易。
3. 资金管理优化
由于无法日内撤单重来，量化策略需要更严谨的资金分配逻辑，确保在不同
交易日之间保持稳定的仓位水平。

停牌复牌：异常波动处理与数据清洗难点
停牌对量化的干扰
无法交易：
策略发出买卖信号却无法执行，导致持仓被动锁定或资金闲置。
数据断裂：
停牌期间OHLCV数据缺失，影响技术指标（如均线）的计算。
复牌跳空：
停牌期间积累的消息在复牌瞬间释放，导致价格剧烈跳空，回测
中易产生“幸存者偏差”。
1. 标的剔除逻辑
在策略运行前，首先通过交易状态位检查。若标的处于停牌状态，则自动忽略该
标的的买入信号，防止无效下单。
2. 数据填充方案
对于停牌期间的缺失数据，通常采用前值填充 (Forward Fill)。价格保持停牌前最后
一天的收盘价，成交量填充为0。
3. 复牌首日处理
复牌首日往往波动巨大且流动性异常。严谨的回测系统会设置复牌首日禁买/禁卖
逻辑，以模拟真实的流动性约束。

分红除权：前复权与后复权的数据处理逻辑
除权除息 (Ex-dividend/Ex-rights)
当公司进行分红或送股时，股价会进行自动下调。如果不进行复权处理，K线图上会出现巨大的价格跳空，导致技术指标失效及收益率计算错误。
前复权 (Adjusted Backward)
逻辑：保持当前价格不变，将历史价格按比例下调。
优点：能够直观看到当前股价，且K线走势连续。
场景：最常用。适用于计算均线、MACD等基于价格的技术指标。
后复权 (Adjusted Forward)
逻辑：保持上市首日价格不变，将后续价格按比例上调。
优点：能够真实反映从上市至今的累计收益。
场景：适用于计算策略的真实净值曲线及回测长期收益率。
量化金律：
策略研究中必须统一复权方式，严禁混合使用原始价与复权价。

隐性成本：滑点产生原因与控制方法
滑点往往是显性成本的 2-5 倍，是吃掉策略利润的“无形杀手”。
为什么会产生滑点？
• 市场冲击： 你的买单太大，扫光了低价档位，被迫在高价成交。
 • 执行延迟： 信号发出到订单抵达交易所的时间差内，价格已变动。
 • 流动性不足： 买卖价差过大，导致成交价偏离预期。
如何有效控制滑点？
算法交易：使用 VWAP / TWAP 将大单拆分为无数小单。
分批下单：避免在同一时间点集中爆发交易。
流动性筛选：策略选股时剔除日均成交额过低的“僵尸股”。
被动成交：尽量使用限价单（Limit Order）而非市价单。

数据处理：严格避免未来信息与前视偏差
核心禁令：严禁使用未来信息
在回测中，如果你在 T 日 使用了 T 日收盘后 才发布的数据（如财报、盘后公告），你就犯了“前视偏差”错误。这会导致回测收益虚高，实盘却无法复现。
严格时间对齐
确保数据的使用时间晚于其发布时间。例如：
2023年年报在2024年4月发布，策略只能在4月
后使用该数据。
数据一致性
建立数据血缘关系，确保因子计算所用的起始
数据来源可靠、逻辑统一，避免计算过程中的
信息泄露。
异常值处理
针对涨跌停、停牌、除权除息等特殊情况进行
清洗，确保模型学习到的是真实的市场规律而
非噪音。

常见数据污染形式
污染类型
现象与危害
解决方案
停牌数据
产生虚假收益缝隙，实盘无法交易。

超额收益 (Alpha) 概念：超越基准的艺术
Alpha = 投资组合收益 - 基准收益
收益曲线对比图 (示意)
为什么 Alpha 更重要？
绝对收益受大盘波动（Beta）影响巨大。Alpha 衡量的是策
略真正的选股/择时能力，是剔除市场噪音后的核心价值。
计算示例
• 组合收益：10%
 • 大盘基准：2%
 • 超额 Alpha：8%
 即使大盘跌 10%，若组合只跌 2%，Alpha 仍为 8%。

流动性因子
成交金额对收益的影响。
BARRA 模型认为：股票收益 = Σ (因子暴露 × 因子收益) + 特异性收益。
因子正交化 (Orthogonalization)
问题：多重共线性
不同因子间存在相关性（如小市值通常伴随高波动），导致模型参数不稳定。
解决方案：线性回归剥离
Factor_New = Factor_Old - (β × Factor_Ref)
# 剥离 Factor_Old 中与 Factor_Ref 相关的部分
结果：正交后的因子互不相关，能准确评估每个因子的独立贡献。

$ uv init quant-project #初始化项目

$ uv run XXXX.py
uv run jupyter lab

本，确保环境 100% 可复现。
极速同步
更换电脑或分享项目时，只需运行 uv sync，数秒
