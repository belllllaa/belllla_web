# 彼得·林奇（Peter Lynch）策略与 A 股量化

## 一、林奇的核心策略要点

### 1. PEG 比率（核心估值指标）

- **公式**：PEG = 市盈率（PE）÷ 盈利增长率（Growth rate，用百分比）
- **含义**：林奇认为「一家定价合理的公司，其 PE 应等于其增长率」；PEG 把成长性纳入估值，比单纯 PE 更合理。
- **经验标准**：
  - **PEG < 0.5**：最理想（成长性好且估值低）
  - **PEG < 1.0**：偏低估，值得关注
  - **PEG > 1.0**：偏贵

### 2. 投资哲学（GARP：Growth at a Reasonable Price）

- 只买自己能理解的公司
- 做好功课，关注基本面（现金流、负债、库存等）
- 在合理价格买成长股，寻找「十倍股」（ten-baggers）
- 六类公司区别对待：**快速成长、稳健型、慢速成长、周期、转机、资产型**；高速成长型重点看 PE 与 PEG

### 3. 其他常用筛选

- 现金流强、负债率 preferably < 50%
- 库存增速不宜明显快于销售
- 业务简单、有品牌/可理解
- 偏好有内部人增持的公司

## 二、在 A 股 / 量化里是否适用？

**结论：适用，且已有不少 A 股量化回测与实盘思路。**

### 1. 与 A 股的匹配点

- PEG 所需数据在 A 股都可获取：PE（或 TTM）、盈利增长率（同比、复合增速等）。
- 林奇强调的「被冷落、基本面好、估值合理」在 A 股同样存在（中小市值、机构覆盖少、业绩拐点等）。
- 实践上常见做法：**剔除 ST、停牌、次新股 → 算 PEG → 按市值/流动性/行业做约束 → 定期调仓**，与当前 QMT 策略思路一致。

### 2. 公开回测/实盘参考（多为 A 股）

| 来源/策略 | 区间/设定 | 年化收益 | 最大回撤 | 备注 |
|-----------|------------|----------|----------|------|
| 东方财富 PEG 价值选股（优化版） | 2019–2022 | **48.25%** | 16.96% | PEG≤0.4，持股约 20 只，相对沪深 300 超额明显 |
| 同文 | 初始版 | 34.67% | 21.12% | 未做参数收紧 |
| BigQuant 低 PEG 稳健增长轮动 | — | **28.34%** | 39.65% | 每月调仓 |
| BigQuant 低 PEG 小市值轮动 | — | 16.96% | 43.94% | 每 5 日调仓 |
| 同花顺 PEG 策略 | 2022.11–2023.10 | 16.14% | 9.63% | 回撤较小 |

共同点：**PEG 选股 + 定期调仓（月/周）+ 控制回撤（仓位/分散/止损或仅时间调仓）**，和你在 QMT 里「按时间调仓、不设止盈止损」的版本在逻辑上同源。

### 3. 量化实现时要注意

- **数据**：PE、EPS、历史利润/营收用于算增长率；若平台无基本面，可用「价格 + 历史涨幅」做 PEG 代理（你当前策略已做）。
- **过滤**：剔除 ST、停牌、上市不足 1 年；可选：PE 为正、增长率为正、至少 N 期财报。
- **调仓**：常见每月初或每 N 日调仓；持股 5–20 只，等权或按市值/因子加权。
- **交易规则**：部分回测会加「涨停不买、跌停不卖」等，可按需要开启。

## 三、与本仓库 QMT 策略的对应关系

- 文件 **`strategies_old/strategy_peter_lynch_peg.py`** 即「林奇 PEG + 仅按时间调仓」的 A 股实现：
  - 用 **PEG < 0.5（或可调）** 筛选；
  - 无真实 PE/增长率时用 **1 年涨幅做增长率代理**，PE 代理固定，保证回测能跑；
  - **仅按调仓日换仓**，不设止盈止损；
  - 可选剔除周期/项目类行业、优先小市值。

若后续接入真实 PE/盈利增长率（如 QMT 扩展接口或外部数据），可把 `get_pe_ratio` / `calculate_earnings_growth_rate` 换成真实计算，其余框架可不变。

## 四、PEG 价值选股（优化版）与当前 QMT 策略的差异

| 项目 | 优化版（约 48% 年化 / 17% 回撤） | 当前 strategy_peter_lynch_peg.py |
|------|----------------------------------|----------------------------------|
| **PEG 门槛** | PEG ≤ **0.4**（更严） | PEG < **0.5**（可调 `peg_proxy_max`） |
| **持股数量** | **20 只**（分散更好） | **5 只**（可调 `max_stocks`） |
| **调仓频率** | **每月月初** | 每 **30 天**（自然日，可调） |
| **次新股** | 剔除上市不满 **1 年** | 未剔除 |
| **交易规则** | **涨停不买、跌停不卖** | 未做 |
| **数据要求** | PE>0、增长率>0、财报≥4 期 | 无基本面时用 1 年涨幅代理 |
| **选股排序** | 市值最小的 N 只 | 已按市值最小排序 ✓ |
| **剔除** | ST、停牌、次新 | ST/创业板/科创板/北交所 ✓ |

优化方向（在 QMT 中可做）：把 PEG 收紧到 0.4、持股提到 20 只、剔除次新股、调仓日涨停不买/跌停不卖、可选「每月首个交易日」调仓。

## 五、同花顺彼得策略对照与可优化点

同花顺上的彼得·林奇策略介绍（投资攻略 A–E）与当前 `strategy_peter_lynch_peg.py` 的对应与可优化方向如下。

| 同花顺要点 | 当前实现 | 说明/优化 |
|------------|----------|-----------|
| **A. PEG < 0.5** | ✓ 已实现 | `peg_proxy_max=0.5`，无 PE 时用 1 年涨幅代理。 |
| **B. ES 风险平价配权** | 未实现 | 当前等权（每只固定金额）。可做：按近期波动率或 ES 反比配权，低波动多配。 |
| **D. 最大 5 只，优先小市值，15 天调仓** | ✓ 已对齐 | 调仓周期 15 天；市值排序取小已实现。 |
| **E. 剔除周期性与项目类行业** | ✓ 已实现 | 用板块反建 股票→行业，再按关键字剔除。 |

**已在代码中实现的优化**：调仓周期 15 天、选股全池扫描后按市值取小、轮动模式。

**后续可做**：B 的 ES/波动率反比配权（每只股买入金额按风险调整）。
