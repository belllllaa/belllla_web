# 脚本说明

## peg_top5_eastmoney.py

**功能**：用东方财富的机构预测（盈利预测一致预期）计算每只股票的 PEG，筛出 **PEG<0.5** 且 **市值最小** 的前 5 只，并打印结果。

- **PEG 定义**：`PEG = PE / 未来三年复合增长率（%）`。例如当前为 2025 年时，使用 2025、2026、2027 三年预测每股收益，计算 **2025→2027 的复合增长率（CAGR）**：`(EPS_2027/EPS_2025)^(1/2)-1`，再×100 得到百分比。
- **PE、市值**：来自东方财富沪深京 A 股实时行情（`市盈率-动态`、`总市值`）。
- **筛选**：只保留 0 < PEG ≤ 0.5 的股票，再按总市值升序取前 5。

**依赖**：

```bash
pip install akshare pandas
```

**运行**（在项目根目录）：

```bash
python scripts/peg_top5_eastmoney.py
```

建议每日收盘后运行一次。若请求行情时出现代理/网络错误，可检查本机代理或稍后重试。
