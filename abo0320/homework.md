## 🟡 进阶作业：研究环境实战（30分钟）

**题目 3：实现 PEG 因子（价值与成长的结合）**
彼得·林奇非常推崇 PEG 指标，认为它修正了 PE 指标不看增长的缺陷。请在研究环境（Notebook）中实现它。

*   **公式**：$$PEG = \frac{PE}{\text{营收增长率(%)}}$$ 
*   **操作步骤**：
    1.  **获取数据**：使用 `get_fundamentals` 获取 `valuation.pe_ratio` 和 `indicator.inc_revenue_year_on_year`。
    2.  **数据清洗**：
        *   剔除 PE < 0 的亏损股。
        *   剔除 增长率 <= 0 的股票（PEG对负增长无意义）。
    3.  **计算 PEG**：直接相除。
    4.  **标准化**：
        *   PEG 是“越低越好”还是“越高越好”？（答案：越低越好，类似PE）。
        *   请对 PEG 进行去极值（Winsorize）和 Z-Score 标准化（记得取负号或调整方向，使其变为“越大越好”以便合成）。
    5.  **选股**：选出 PEG 评分最高的 20 只股票。
