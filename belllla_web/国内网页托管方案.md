# 国内可访问的网页托管方案

以下平台在中国大陆**无需翻墙**即可访问，适合分享给国内朋友。

---

## 方案一：腾讯云 CloudBase（推荐）

**网址**：https://cloud.tencent.com/product/tcb  
**免费额度**：1GB 存储 + 5GB 流量/月

**步骤**：
1. 注册腾讯云账号
2. 开通「云开发 CloudBase」
3. 创建静态网站托管
4. 上传 `lishui-trip.html`（重命名为 index.html）或整个文件夹
5. 获得访问链接，如 `https://xxx.tcloudbaseapp.com`

**优点**：国内 CDN 加速，访问快，免费额度够用

---

## 方案二：腾讯云 COS 对象存储

**网址**：https://cloud.tencent.com/product/cos  
**免费额度**：新用户有免费额度

**步骤**：
1. 注册腾讯云
2. 创建 COS 存储桶
3. 开启「静态网站」功能
4. 上传 index.html（即 lishui-trip.html）
5. 使用自带访问域名

**优点**：稳定，适合长期使用

---

## 方案三：Coding.net（腾讯云开发者）

**网址**：https://coding.net  
**免费**：个人项目免费

**步骤**：
1. 注册 Coding 账号
2. 创建项目，上传代码
3. 开启「持续部署」→「静态网站」
4. 部署后获得访问链接

**优点**：类似 GitLab，支持 Git 推送自动部署

---

## 方案四：Gitee 码云 Pages

**网址**：https://gitee.com  
**说明**：Gitee Pages 服务可能有变动，需实名认证

**步骤**：
1. 注册 Gitee
2. 创建公开仓库
3. 上传 lishui-trip.html
4. 服务 → Gitee Pages → 启动

**注意**：需确认当前是否仍提供 Pages 服务

---

## 方案五：直接发文件（最简单）

无需任何托管，直接把 `lishui-trip.html` 发给对方：
- 微信 / QQ 发送文件
- 对方用浏览器打开即可（需联网加载地图）

---
